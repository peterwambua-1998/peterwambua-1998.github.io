AI Agent Builder Platform | React (TypeScript) · NestJS · LangGraph

REACT REACT FLOW LANGCHAIN RAG NODE.JS AWS S3 POSTGRES (vector extension for storing embeddings) Passport for authentication Stripe , Paystack and PayPal

A full-stack platform that enables users to design, configure, and deploy custom AI agents for business automation tasks including sales operations, lead generation, and customer support. Built a React/TypeScript frontend with a NestJS backend structured as a monorepo, integrating LangGraph for orchestrating multi-step agentic workflows. The system supports voice/call recording capabilities and social media interactions, with a modular architecture that allows agents to be composed and extended for diverse use cases.

KEY FEATURES:

  • Visual flow builder for creating complex conversation paths
  • AI RAG (Retrieval Augmented Generation) capability
  • Knowledge base integration via document upload, direct text input, or website crawling
  • Image processing and recognition
  • Embeddable widget for easy website integration
  • Integrate bot with WhatsApp, Instagram and Facebook
  • Conversation management

TECH STACK:

React, React Flow, Node.js, Langchain, Postgres, Pg-vector, AWS S3
