AI CHATBOT BUILDER

REACT REACT FLOW LANGCHAIN RAG

A no-code AI chatbot builder platform where users can configure intelligent conversation flows using a visual interface powered by React Flow. The project is still in development phase.
This project really tested my programming skills especially in data structures and algorithms. I also learned new technologies like ffmpeg to convert files, useful for converting audio files to wav that iis best suited for googles speech to text.

KEY FEATURES:

  • Visual flow builder for creating complex conversation paths
  • AI RAG (Retrieval Augmented Generation) capability
  • Knowledge base integration via document upload, direct text input, or website crawling
  • Image processing and recognition
  • Embeddable widget for easy website integration
  • Integrate bot with whatsapp, instagram and facebook
  • Conversation management

TECH STACK:

React, React Flow, Node.js, Langchain, Postgres, Pg-vector, AWS S3
